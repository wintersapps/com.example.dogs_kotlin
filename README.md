# com.example.dogs_kotlin

Sample Android application displaying dog breeds in the MVVM architecture.

Technologies and solutions used:

- Material Design
- Navigation
- LiveData
- Retrofit
- RxJava
- RxAndroid
- Glide
- Room
- Coroutines
- Shared Preferences
- Data Binding
- View Binding
- Palette
- Notifications
- Android Preferences
- Android Permissions (SEND_SMS)
- Multidex
